# Kova Studio — Ecommerce Measurement Implementation

A Shopify development store simulating a D2C fashion brand (Kova Studio). Implements the full GA4 enhanced ecommerce event funnel, Google Ads conversion tracking with enhanced conversions, server-side GTM via Stape.io, cross-domain checkout tracking, and Consent Mode v2.

→ **[View project case study on GitHub Pages](https://dexhamter.github.io/google-ads-measurement-library/projects/02-ecommerce/)**
